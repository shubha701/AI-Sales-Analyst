# 📊 AI Sales Analytics Dashboard

An AI-powered Sales Analytics Dashboard built with **Streamlit**, **Plotly**, and **Google Gemini AI**. This application enables businesses to upload sales datasets, visualize key performance metrics, generate AI-driven business insights, and export reports through an intuitive and interactive web interface.

---

## 🚀 Features

### 🔐 User Authentication
- Login and Signup system
- Session-based authentication
- Secure user access

### 📂 Dataset Upload
- Upload CSV, XLS, or XLSX files
- Automatic data cleaning
- Missing value handling
- Duplicate record removal

### 📊 Interactive Dashboard
- Revenue KPI
- Orders KPI
- Units Sold KPI
- Average Order Value
- Executive Dashboard
- Interactive Charts

### 📈 Advanced Analytics
- Revenue by Product
- Revenue by Region
- Monthly Revenue Trend
- Product Performance Analysis
- Revenue Distribution
- Business Insights

### 🤖 AI Business Analyst
Powered by **Google Gemini AI**

Ask questions such as:
- Which product generates the highest revenue?
- Which region performs best?
- What are the sales trends?
- Give strategic recommendations.
- How can revenue be improved?

The AI provides:
- Direct Answers
- Key Findings
- Business Insights
- Strategic Recommendations
- Executive Summary

### 📋 Executive Summary Generator
Automatically generates professional business summaries including:
- Performance Overview
- Key Findings
- Risks
- Opportunities
- Recommendations
- Final Conclusion

### 📥 Reports Center
- Download CSV Reports
- Download Excel Reports
- Dataset Preview
- Business Highlights
- Dataset Statistics

# 📦 Installation

## Clone the Repository

```bash
git clone https://github.com/your-username/AI-Sales-Analytics-Dashboard.git

cd AI-Sales-Analytics-Dashboard
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Create Environment File

Create a `.env` file in the project directory.

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

Replace `YOUR_GEMINI_API_KEY` with your Google Gemini API key.

---

## Run the Application

```bash
streamlit run app.py
```
